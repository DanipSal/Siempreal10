## 🤖 AFIP Automation Tool (Selenium IDE)
I developed a specialized automation script using **Selenium IDE** to streamline the upload of tax documents (comprobantes) directly into the **AFIP** (Argentina's Federal Administration of Public Revenue) official portal.

![Selenium AFIP Automation](https://i.postimg.cc/WbPc4LND/Demo-Selenium-IDE.png)

* **Key Achievement:** Fully automated the repetitive data entry process in the AFIP portal, significantly reducing processing time and human error.
* **Techniques Used:** Element identification, automated form filling, and navigation flow control.
* **Impact:** Transformed a manual administrative task into an efficient automated workflow.
